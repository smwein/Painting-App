# Design: 3.5.26 #1 Notes (6 Changes)

## 1. Delete Entire Sections on All Settings Pages

All 5 settings pages (Simple Interior, Simple Exterior, Interior Detailed, Exterior Detailed, Per Room) get a delete button on every section card. The store's `deleteSection` already cascades to remove child line items. For the detailed settings pages, default sections (like "Walls", "Trim") will be deletable too -- the `isDefault` flag will no longer prevent deletion from settings UI. Simple Interior/Exterior already have section delete on custom sections; the detailed pages need it added to each section header.

## 2. Drag-and-Drop ALL Sections on Interior/Exterior Detailed Settings

Wrap all cards in both `InteriorDetailedPricing.tsx` and `ExteriorDetailedPricing.tsx` in a `SortableSettingsCard` + `DndContext` pattern (same as Simple settings pages). Each page gets a section order array persisted to `settingsPageSectionOrder.interiorDetailed` and `settingsPageSectionOrder.exteriorDetailed`. Section IDs for Interior Detailed: `int-multipliers`, `int-furnished-rates`, `int-empty-rates`, plus all dynamic section IDs, `int-modifiers`, `int-paint-prices`. Same pattern for Exterior.

## 3. Allow Deleting Any Line Item on Settings Pages

Currently all dynamic line items show delete buttons. The built-in rate inputs (Wall Sqft, Ceiling Sqft, Trim LF) on furnished/empty rate cards are hardcoded `<Input>` fields, not dynamic line items. To make them deletable without a major refactor, add a "hidden line items" feature using the existing `hiddenLineItems` array -- an X button next to each built-in input that hides it from both settings and calculator pages.

## 4. Remove Section Delete from Calculator Pages

Remove section delete buttons (red X) from `InteriorDetailed.tsx` and `ExteriorDetailed.tsx` calculator components. Keep room remove button on Per Room calculator (removing a room from an estimate is an estimating action, not admin config).

## 5. Estimated Paint + Coverage Rates for Per Room

Add new `perRoomCoverage` field to `PricingSettings`: `{ wallSqftPerGallon, ceilingSqftPerGallon, trimLfPerGallon }` with defaults matching interior coverage. Add editable inputs on Per Room settings page. On the Per Room calculator, add an "Estimated Paint" yellow card (same style as other calculators) that sums wall/ceiling/trim across all rooms and divides by coverage rates.

## 6. Total Measurements Grid on Per Room Calculator

Add a "Total Measurements From All Rooms" card on the Per Room calculator that sums all rooms' wallSqft, ceilingSqft, and trimLF into a 3-column grid display (same blue card style as auto-calculated measurements on other calculators).
